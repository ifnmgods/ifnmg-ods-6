Como Funciona

Notificações Automáticas:
• Quando alguém envia uma mensagem de suporte, você recebe um email imediatamente
• Quando você responde uma mensagem, o usuário recebe sua resposta por email
• Uma cópia da resposta também é enviada para você

Configuração:
• Para usar em produção, configure um email real nas variáveis de ambiente
• Recomendamos usar um email dedicado para o sistema
• Configure a senha de aplicativo para maior segurança

Teste:
Use o botão "Testar Email" para verificar se as notificações estão funcionando corretamente.
